## Features

* **Ping** — check host reachability and round-trip latency
* **Whois** — look up domain and IP registration details
* **Traceroute** — map the route packets take to any host
* **DNS Lookup** — resolve records for any domain
* **Port Scanner** — discover open ports on a host
* **IP Calculator** — compute subnets, masks, and ranges
* **Connection Log** — track active network connections
* **Router Setup tool** — quick access to your router configuration
* **IP & Host Converter** — convert between IPs and host names
* **WiFi Analyzer & LAN Scanner** — inspect nearby networks and discover devices on your LAN
* **Network statistics** — live traffic and interface stats
* And more...

## Use cases

## Installing the APK

1. Download the latest `*.apk` from the [Releases page](https://github.com/IP-Tools-App/ip-tools-apk/releases/latest).
2. On your device, allow installation from unknown sources (Settings → Apps → Special access → Install unknown apps) for your browser or file manager.
3. Open the downloaded APK and tap **Install**.
4. If you previously installed IP Tools from Google Play, **uninstall it first** — the Play Store build is signed by Google and the APK won't install over it.

## FAQ

**Is IP Tools free?**
Yes. You can download and use it for free.

**Why does the APK fail to install?**
You most likely have the Google Play version installed. Uninstall it first — the two builds use different signing keys and Android won't let one replace the other.

**Do I need root?**
No. IP Tools works on standard, non-rooted devices.

**Which should I download — APK or Google Play?**
Google Play is recommended for automatic updates. Use the APK if you can't access Play or want a specific version.
